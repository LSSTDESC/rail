*****************
Interactive Usage
*****************

.. introduction of what interactive is, note that it's a front end to functionality, when you should or shouldn't use, link to notebooks

RAIL can be run interactively, such as in a Jupyter notebook. This mode of
operation is called the interactive mode. While the underlying framework doesn't
change, the interactive mode provides a front-end to RAIL functionality.
